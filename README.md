Character Counter

This project is a simple **Character Counter Web Application** built using **HTML, CSS, and JavaScript**.
It allows users to type text in a textarea and displays the number of characters typed along with the remaining characters in real time.

Live Demo
https://harish-20cs022.github.io/Character-Counter/

Features

- Real-time character counting
- Displays typed characters and remaining characters
- Maximum character limit (200 characters)
- Simple and clean user interface
- Instant updates while typing

Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

How It Works

- The user types text inside a **textarea**
- JavaScript listens for the **input event**
- The program counts the number of characters entered
- The counter updates automatically on the screen
